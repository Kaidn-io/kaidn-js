Kaidn Playground

Try the Kaidn client libraries locally, with your own keys. Two tools:

🎯 Score playground

Run /v1/score through @kaidn/sdk (server-side). Fill an event, get the verdict + reasons, and copy the code. Uses your secret API key.

🔎 Fingerprint test

Run @kaidn/fp in this browser — see your device_id and automation signals, and optionally beacon them to capture a JA4. Uses a publishable key.

Set keys in a .env file (copy .env.example) or paste them in the pages. See the repo README for the full guide.
